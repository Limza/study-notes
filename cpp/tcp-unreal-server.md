C++ TCP 서버와 Unreal 클라이언트 연동을 학습한 레포입니다.

## 정리

- Unreal 클라이언트와 외부 TCP 서버 연결
- 메시지 송수신 흐름
- 클라이언트 입력과 서버 상태 처리의 경계
- 네트워크 지연, 끊김, 재접속 상황
- 게임 클라이언트와 서버를 분리해서 생각하는 방식

Unreal 클라이언트와 C++ TCP 서버를 분리해서 연동해보며,
