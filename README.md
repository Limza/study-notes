# Study Notes

공부한 내용을 한곳에서 훑어보기 위한 기술 노트 허브입니다.

큰 코드 저장소를 그대로 합치기보다, 이 저장소에는 요약, 배운 점, 다시 확인할 포인트, 원본 레포 링크를 정리합니다.

## 분류

- `cpp`: C++, TCP 서버, Conan 패키지 관리, Unreal 연동 서버 학습
- `unreal`: Unreal Engine, Dedicated Server 관련 학습
- `reading`: 책을 읽고 기술 학습 방식에 적용한 메모

## 정리 방식

- 코드와 에셋이 큰 레포는 원본 저장소에 둔다.
- `study-notes`에는 핵심 요약과 링크만 둔다.
- 새로 공부한 내용은 먼저 노트로 정리하고, 코드 예제가 커지면 별도 레포로 분리한다.
- 나중에 다시 봐도 흐름을 떠올릴 수 있도록 문제, 선택, 시행착오를 남긴다.
